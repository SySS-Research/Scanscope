Bubble Chart View

How to use the Bubble Chart:
  • Each bubble represents a group of hosts with identical port configurations
  • Bubble size indicates the number of hosts in that group
  • Similar port configurations are positioned close together
  • Hover over bubbles to see port details and host count
  • Click bubbles to select and view detailed host information below
  • Use pan and zoom tools to explore the chart
  • Select multiple bubbles with box or lasso selection
